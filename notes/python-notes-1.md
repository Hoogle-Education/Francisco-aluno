# Variáveis

|Tipo|Objetivo|Atalho|
|:-:|:-:|:-:|
|Inteiro|Guardar números sem ponto flutuante|`int`|
|Real|Guarda números com ponto flutuante|`float`|
|Frase|Guarda uma sequencia de caracteres|`str`|
|Booleano|Guarda verdadeiro ou falso|`bool`|
